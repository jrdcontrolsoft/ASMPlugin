ASM Plug-In - ASM Editor

Here functions of editor or IDE are shown if an ASM file is open.


  1. Select ASM file.
  2. All results of the compiling and link process are contained in build directory.
  3. Open ASM file in the editor.
    Changing between text and comment can be concluded with [CTRL]+[7] or [CTRL]+[SHIFT]+[7].
  4. The content assist can be reported with [CTRL]+[SPACE].
  5. The output console contains information about the compiling and link process.
  6. The outline contains links to procedures, macros, labels and segments.
