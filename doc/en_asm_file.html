ASM Plug-In - New ASM File

In this dialog you can choose whether you want to cause an ASM project or an ASM file.


  1. Choice to the extended new-dialog.
  2. Making new ASM project.
  3. Making new ASM file.
  4. Next to set attributes of selected entry.

In this dialog you can set file name of the new ASM file.


  1. The choosed project where be file added.
  2. Filename of new ASM file
  3. Next to set more attributes.
  4. Finish and create new ASM file.

In this dialog you can create new ASm file from a template.


  1. Enable template list.
  2. List contains all templates.
  3. Finish and create new ASM file.
