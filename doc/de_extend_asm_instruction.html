ASM Plug-In - ASM Befehle erweitern

Sie haben die möglichkeit weitere ASM Befehle zu ergänzen. Wechseln Sie dazu in das ASM Plug-In-Verzeichnis ([ECLIPSE-INSTALLATIONSVERZEICHNIS]\plugins\de.fh_zwickau.asmplugin_1.0.0). Dort befindet sich die Datei: asm_instruction_set.xml . Diese XML Datei enthählt alle vom System verwendeten Befehle.

Auszug aus der asm_instruction_set.xml :
<?xml version="1.0" encoding="iso-8859-1"?>
<sets>
  <instruction command="AAA" description="ASCII Adjust after Addition" />
  <instruction command="AAD" description="ASCII Adjust AX before Division" />
  <instruction command="AAM" description="ASCII adjust alter multiply" />
  <segment field=".DATA" description="Assembler directive that defines program variables." />
  <segment field=".CODE" description="Assembler directive that defines the program instructions." />
