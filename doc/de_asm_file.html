ASM Plug-In - Neue ASM Datei

In diesem Dialog können Sie festlegen, ob Sie ein ASM Projekt oder eine ASM Datei erzeugen wollen.


  1. Auswahl zum erweiterten Neu-Dialog zu kommen.
  2. Neues ASM Projekt erstellen.
  3. Neue ASM Datei erstellen.
  4. Weiter um die Eigenschaften des gewählten Eintrags festzulegen.

In diesem Dialog können Sie unteranderem den Dateiname der neuen ASM Datei festlegen.


  1. Das gewählte Projekt zu dem die neue ASM Datei hinzugefügt werden soll.
  2. Dateiname der neuen ASM Datei.
  3. Weiter um weitere Eigenschaften festzulegen.
  4. Abschließen und neue ASM Datei erstellen.

In diesem Dialog können Sie eine neue ASM Datei aus einer Vorlage erzeugen.


  1. Aktiviert die Vorlagen-Liste.
  2. Liste aller vorhandener Vorlagen.
  3. Abschließen und neue ASM Datei erstellen.
